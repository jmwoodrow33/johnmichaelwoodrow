# John-Michael Woodrow Portfolio

A personal portfolio website showcasing software development projects and professional experience.

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Custom CSS with responsive design
- **Routing**: React Router
- **Deployment**: GitHub Pages

## Features

- Responsive navigation with mobile drawer
- Project showcases with videos and descriptions
- Contact information and links
- Dark theme optimized for readability

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site is configured for GitHub Pages deployment. The build output in `dist/` can be deployed directly.
